# Github Explorer
Um buscador de repositórios do Github que também permite explorar a suas issues ([ver](/layout-issues.png)).

### Projeto desenvolvido com:

* TypeScript
* React
* React Hooks
* React Router Dom
* Axios
* Styled Components
* Local Storage
